# InterviewAI

AI-powered mock interview platform built using MERN stack and OpenRouter API.

## Features

- Role-based interview generation
- Technical & behavioral interviews
- AI-generated interview questions
- Interactive conversational sessions
- JWT authentication
- Interview history tracking

## Tech Stack

Frontend:
- React.js

Backend:
- Node.js
- Express.js

Database:
- MongoDB

AI:
- OpenRouter API

## Architecture

User
↓
React Frontend
↓
Express Backend
↓
OpenRouter API
↓
MongoDB
